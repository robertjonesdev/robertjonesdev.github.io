Console-based RPG

“Myrddin Emry’s Realm” is a single-player, text-based role playing game developed in C++ using object-oriented programming concepts. The character, Myrddin Emrys, starts in a virtual, mapped world. The player can move north, south, east or west through the map, depending on available paths. Myrddin will encounter locked doors, items, healing potions, and eventually a Red Dragon. To win, the player must avoid death and defeat the Red Dragon.

C++ & OOP Concepts Implemented

  • Classes and Objects
  • Inheritance and Polymorphism
  • Dynamic Memory Allocation
  • File Input
  • Vectors
  • Hash Map
